# HelloToast Assignment 1

## Overview
This project is an enhanced version of the HelloToast Android application for Mobile App Development 1.

The app demonstrates basic Android UI design and button interactions.

## Features Implemented

- **Toast Button**
    - Displays a short toast message when clicked.

- **Count Button**
    - Increases the counter value by 1.
    - Changes its background color dynamically:
        - Green for odd numbers.
        - Blue for even numbers.

- **Zero Button**
    - Resets the counter value to 0.
    - Starts with a gray background.
    - Changes to orange after pressing Count.
    - Returns to gray after resetting.

## Layout Enhancements

- Added a new **Zero** button between Toast and Count.
- Arranged the three buttons vertically.
- Aligned buttons on the left side of the counter TextView.

## Responsive Design

The application supports both:
- Portrait mode
- Landscape mode

The layout remains functional and responsive in both orientations.

## Technologies Used

- Java
- XML
- Android Studio
- ConstraintLayout

## How to Run

1. Clone or download the project.
2. Open it in Android Studio.
3. Sync Gradle files.
4. Run on emulator or Android device.
